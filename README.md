# Parent-Teacher Communication System

## 📌 Description

This is a web-based application developed using Spring Boot and MySQL to improve communication between parents and teachers.

## 🚀 Features

* Admin can manage students and teachers
* Teachers can mark attendance and upload marks
* Parents can view attendance, marks, and messages

## 🛠️ Technologies Used

* Java
* Spring Boot
* MySQL
* HTML, CSS, JavaScript

✅ Completed (College Project)
